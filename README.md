# Real Estate Listing Platform

A full-stack web application that allows users to browse, list, and manage real estate properties.  
The platform provides a seamless experience for buyers, owners, and agents to interact in a secure environment.

---

## Tech Stack

### Frontend
- React.js
- HTML5
- CSS3
- JavaScript

### Backend
- ASP.NET Core (Microservices / REST APIs)
- Spring Boot (Java-based backend services)
- J2EE (Enterprise application components)

### Database
- MySQL

### Tools & Technologies
- Git & GitHub
- AWS
---

##  Key Features

User Authentication & Authorization  
Property Listing & Management  
Advanced Property Search & Filters  
Responsive UI for all devices  
Secure API Integration  
Role-based Access (Admin / Buyer / Agent / Owner)
