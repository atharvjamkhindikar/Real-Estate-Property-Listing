# Database

This folder contains all database-related resources for the **Real Estate Listing Platform**.

It documents the database design, structure, and scripts used by the backend services.

---

##  Includes
- Database schema design
- ER diagrams (conceptual & implementation-level)
- SQL scripts (DDL)
- Table creation queries
- Relationships and constraints
- Indexing strategies for performance optimization

---

## ER Diagrams
- **Conceptual ERD**  
  High-level representation of entities and their relationships (Users, Properties, Property Images, etc.)

- **MySQL Workbench ERD**  
  Auto-generated from the actual database schema, showing tables, columns, data types, primary keys, and foreign keys.

Both diagrams are included for better understanding and documentation.

---

## Database Type
- **Relational Database**
- **MySQL**
